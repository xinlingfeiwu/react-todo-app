🔧 技术修复详情

  • 唯一ID前缀：每个SVG使用如 ios180_、android192_ 的前缀
  • 保持viewBox：使用原始 viewBox="0 0 64 64" 保持比例
  • 正确尺寸：只修改 width 和 height 属性
  • 避免transform：不使用 transform="scale()" 避免坐标问题
